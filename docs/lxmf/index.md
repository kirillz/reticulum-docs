# Основы LXMF

## Что такое LXMF?

**LXMF (Lightweight Extensible Message Format)** — это протокол обмена сообщениями, работающий поверх Reticulum. Если RNS доставляет пакеты данных между узлами, то LXMF придаёт этим данным смысл, предоставляя стандартизированный формат для обмена текстами, изображениями, файлами и другим контентом между различными клиентскими приложениями.

### Ключевые особенности

| Особенность | Описание |
|-------------|----------|
| **Совместимость приложений** | Сообщения, отправленные из одного клиента (например, MeshChat), могут быть получены в другом (Columba, Sideband) |
| **Мультимедиа** | Поддержка текста, изображений, файлов и других типов контента в едином формате |
| **Отложенная доставка** | Сообщения хранятся на Propagation-серверах и доставляются адресату при появлении в сети |
| **Асинхронность** | Работает как электронная почта: отправитель и получатель не обязаны быть онлайн одновременно |

---

## Как это работает?

LXMF строится поверх RNS, добавляя уровень структурированных сообщений. Каждое сообщение имеет заголовок с метаданными (отправитель, получатель, timestamp, тип контента) и тело с полезной нагрузкой.

<div class="diagram-center">

```
┌─────────────┐      ┌─────────────────┐      ┌─────────────┐
│  Отправитель│ ──►  │ Propagation     │ ──► │  Получатель │
│  (Sideband) │      │ Server          │      │  (Columba)  │
└─────────────┘      └─────────────────┘      └─────────────┘
        │                      ▲                      │
        │         LXMF Message │                      │
        └──────────────────────┴──────────────────────┘
```

</div>

**Propagation-серверы** играют ключевую роль: они хранят сообщения для офлайн-получателей и доставляют их, когда те появляются в сети. Это делает LXMF идеальным для нестабильных и прерывистых соединений.

## Клиентские приложения

LXMF используется различными клиентами для обмена сообщениями:

| Приложение | Описание |
|------------|----------|
| **Sideband** | Мобильный клиент для Android/iOS с поддержкой мультимедиа |
| **Columba** | Десктопный клиент для Linux/Windows/macOS |
| **MeshChat** | Веб-интерфейс и CLI-клиент для быстрого обмена сообщениями |
| **Nomad Network** | Децентрализованный клиент с поддержкой форумов и каналов |
