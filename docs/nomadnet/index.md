# Nomad Network

**Nomad Network** — это децентрализованный клиент для общения, работающий поверх LXMF и Reticulum. Он предоставляет возможности для обмена сообщениями, участия в форумах и подписки на каналы — всё это без центрального сервера.

## Особенности

- **Децентрализация** — нет единого сервера, все узлы равноправны
- **Анонимность** — не требуется регистрация или привязка к личности
- **Форумы и каналы** — поддержка публичных обсуждений и новостных лент
- **Работа в офлайн-сетях** — функционирует в изолированных mesh-сетях
