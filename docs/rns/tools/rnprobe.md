### rnprobe
`rnprobe` — утилита для проверки доступности и времени отклика узлов Reticulum (аналог ping в IP-сетях).

#### Использование
rnprobe [опции] <destination_hash>

Основные аргументы
<destination_hash> — Hex-хеш пункта назначения (32 байта), который вы хотите проверить.
-c, --config — Путь к директории конфигурации RNS (если нестандартный).
-t, --timeout — Таймаут ожидания ответа в секундах.
## Примеры
Проверка доступности узла
Чтобы проверить связь с узлом, выполните:

rnprobe d2b8215e9d...
Пример вывода
```
[2023-10-27 12:00:00] [VERBOSE] Probe sent to d2b8215e9d...[2023-10-27 12:00:00] [INFO] Valid reply received from d2b8215e9d...Round-trip time: 145 msPath MTU: 500 bytes
```
