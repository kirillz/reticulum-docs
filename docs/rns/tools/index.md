# Утилиты RNS

В этом разделе описаны утилиты командной строки для работы с Reticulum.

## Список утилит

| Утилита | Назначение |
|---------|------------|
| [`rnsd`](rnsd.md) | Демон Reticulum — фоновая служба для работы сети |
| [`rnstatus`](rnstatus.md) | Просмотр статуса сети и подключений |
| [`rnid`](rnid.md) | Генерация и управление идентификаторами |
| [`rnpath`](rnpath.md) | Диагностика маршрутов между узлами |
| [`rnprobe`](rnprobe.md) | Проверка доступности узлов и каналов |
| [`rncp`](rncp.md) | Копирование файлов через сеть Reticulum |
| [`rnx`](rnx.md) | Удалённое выполнение команд на других узлах |
| [`rnsh`](rnsh.md) | Утилита для организации удалённого доступа |
| [`rnodeconf`](rnodeconf.md) | Настройка TNC и радиомодемов RNode |
